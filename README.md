# Reversi-Team2

## Requirements


* `pip3 install --upgrade pip`
* `pip3 install -r requirements.txt`
* `pip3 install mysql-connector-python`
  or manually download here: https://dev.mysql.com/downloads/connector/python/
* I'm using python 3.9.10
